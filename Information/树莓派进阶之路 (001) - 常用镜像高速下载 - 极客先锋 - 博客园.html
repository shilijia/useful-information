树莓派官网在国内打开速度比较慢，想要下载镜像可能需要等待较长的时间，为节约下载镜像的时间，我这里整理了树莓派常用的镜像，在本页面列出了下载地址以及磁力链接，可以更快速的下载镜像，为大家节约时间，同时，镜像也会不断的更新，如果大家有什么好的建议或者疑问，都可以在下面提出来。

 

1. Raspbian

Raspbian是默认的最常用于Raspberry Pi的免费操作系统。Raspbian是基于Debian的一个版本（针对Raspberry Pi硬件Debian进行了专门的优化，并构建了超过35000个Raspbian包）。

Raspbian仍处于积极开发状态，并把重点放在提升尽可能多Debian包的稳定性和性能。对于初学编程的人来说这是一个很好的起点，Raspbian自带的x windows，因此可以使用它类似WIN风格的GUI界面，Raspbian还包括一个“Pi商店”，你可以免费或付费下载一些应用，比如Libre Office、Free Civ（游戏）等等。

2、Ubuntu MATE for the Raspberry Pi 2

 

Ubuntu MATE是桌面Linux发行，其宗旨是通过MATE这个经典、传统的桌面环境来提供Ubuntu操作系统的简介和典雅。MATE是GNOME 2桌面环境的继续，曾经作为Ubuntu的缺省桌面，直到10.10版中被Unity所取代。

Ubuntu MATE适合树莓派新手使用，界面是最好看的，但是在CPU优化方面不如官方的系统做得好。

7、PiNet

Pinet是什么？
它是一个自由和开放源码的项目，为帮助学校建立和管理一个Raspberry Pi的课堂。
其主要特点包括
•基于网络的用户帐户
-基于网络的操作系统-所有树莓PIS启动一个主Raspbian操作系统。
•共享文件夹-易于使用共享文件夹系统的教师和学生。
工作收集系统简单的工作收集和提交系统，让学生在工作。
•自动备份-自动备份所有学生的工作，定期向外部驱动器。
•多个小的功能，如批量用户导入，课堂管理软件集成等
服务器软件安装在运行Ubuntu Linux 14.04计算机（这也完全是免费的）。你必须再连接服务器和覆盆子PIS通过有线网络。

Installing

Once you have the needed equipment, you will need to install Ubuntu 14.04 onto your server computer and then install PiNet. Overall this takes roughly 2-3 hours. Of that 2-3 hours, you are required at the computer for 20-30 minutes.
Finally when the installation is complete you must copy the generated SD card boot files to a blank SD card.

10、NOOBS

NOOBS系统不仅可以非常方便的让一张空白的SD卡变为安装了Rasbian系统的启动盘，而且还可以预包装其他可选的树莓派操作系统，比如Pidora（基于Fedora的系统）、RISC OS 、Arch（Arch Linux的树莓派版） ，甚至像RaspBMC和OpenELEC这样的XMBC系统也不在话下。

并且NOOBS支持多系统启动，即你可以在SD卡上安装多个操作系统，然后再树莓派启动的时候可以从安装的多个树莓派系统中选择你要启动的系统。

在NOOBS安装向导中，共有7种可选的操作系统供我们安装。分别是

  • Rasbian（标准的Debian移植版）
  • Rasbian（轻量级的，面向初学者的编程平台，在树莓派中很流行）
  • Arch Linux（自定义程度高的Linux系统）
  • Pidora（Fedora的分支，对树莓派做了优化）
  • OpenELEC（一种XBMC系统的分支）
  • RaspBMC（一种XBMC系统的分支，专门针对树莓派做了调整）
  • RISC OS（一个轻量级的基于Acorn的操作系统，在80-90年代非常活跃，专门针对树莓派做了优化）

5月29日，sabayon  发布ARM版镜像，可以支持运行在树莓派上2或者3上，最新的版本为Sabayon 16.06 RaspberryPi，镜像基于linux4.4内核，并且包括了可以更新树莓派内核和固件的 rpi-update 工具。
